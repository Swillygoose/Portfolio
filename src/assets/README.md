# CV PDF Files

This folder should contain Sandra Wilmann's CV in both Norwegian and English.

## Required Files

Please add the following PDF files to this folder:

1. **cv-norwegian.pdf** - Sandra Wilmann's CV in Norwegian
2. **cv-english.pdf** - Sandra Wilmann's CV in English

## How to Add PDF Files

Since this is a web application running in Figma Make, you'll need to:

1. Export your CVs as PDF files
2. Name them exactly as:
   - `cv-norwegian.pdf`
   - `cv-english.pdf`
3. Place them in this `/assets` folder

## File Naming

The download buttons on the CV page are configured to save the files as:
- Norwegian: `Sandra_Wilmann_CV_Norwegian.pdf`
- English: `Sandra_Wilmann_CV_English.pdf`

These are the filenames that will appear when users download the CVs.

## Note

If you're deploying this to a web server, make sure the `/assets` folder is publicly accessible and the PDF files are uploaded to the server.
